# GDJ77 Semi Project - Grafolio 모작

## 프로젝트 개요

이 프로젝트는 [OGQ Grafolio](https://grafolio.ogq.me/)의 UI와 UX를 비슷하게 구현하는 것에 중점을 두었습니다.

### 사용 기술
- **개발환경**: STS, Notepad++
- **프론트엔드**: HTML5, CSS, JS, jQuery, Bootstrap
- **백엔드**: Java, Spring Legacy
- **데이터베이스**: Oracle SQL Developer
- **서버**: Tomcat
